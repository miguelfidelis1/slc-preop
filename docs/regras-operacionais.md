# Regras operacionais

## Filtro

```txt
Impresso = Todos
```

## Imprimível

```txt
Data Entrega não contém Pedido pendente
```

E:

```txt
Data Impressão = Não foi impresso
```

ou:

```txt
Data Impressão tem 7 dias ou mais
```

## Ignorar

```txt
Pedido pendente
Pedido impresso recentemente
Pedido sem checkbox
Pedido com data inválida que não seja "Não foi impresso"
```
